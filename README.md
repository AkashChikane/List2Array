# List2Array

List2Array Is extension to simple convert raw list seprated by line break to array for programming.

### Data Types Supported
- Integer
- String
## Features

Select raw list you want to convert into and launch List2Array :)
- Select
- Launch
- Array Returned !!!


## Supported Langauges
1. java
2. javascript
3. python
